# -Mein-Lernportfoli
m Rahmen meiner Umschulung zur Fachinformatikerin für Systemintegration habe ich bereits folgende Themen gelernt:

### 🏢 Lernfeld 1: Unternehmen & Rolle
- Duales System (Betrieb, BBS, IHK)
- Rechte und Pflichten (BBiG, JArbSchG)
- Unternehmensstrukturen (GmbH, UG)
- Geschäftsprozesse und Wertschöpfungskette
- SMART-Methode

### 💻 Lernfeld 2: Arbeitsplatz & Hardware
- EVA-Prinzip
- CPU, RAM, Mainboard
- HDD, SSD, NVMe
- Lastenheft vs. Pflichtenheft
- Arbeitssicherheit

### 🌐 Lernfeld 3: Netzwerktechnik
- Netzwerke (LAN, WAN, VPN, VLAN)
- Router, Switch, Access Point
- IPv4 / IPv6, MAC-Adressen
- Ports (z. B. HTTP, DNS)
- Client-Typen

### 🔐 Lernfeld 4: IT-Sicherheit & Datenschutz
- CIA-Triade (Vertraulichkeit, Integrität, Verfügbarkeit)
- DSGVO, BDSG
- Backup-Strategien
- RAID-Systeme
- IT-Grundschutz

### 📊 Lernfeld 5: Software & Daten
- Zahlensysteme (Binär, Hex, Dezimal)
- Softwarearten
- ERP, CRM, DMS
- Cloud (IaaS, PaaS, SaaS)
- Softwareentwicklung
